    2. Click "Criar conta" — go to registration page

    4. Check browser DevTools > Network: POST to API should return 201
    5. Go to login page, enter credentials
    6. Check DevTools > Network: POST /api/v1/auth/login should return 200 with Set-Cookie header containing "refreshToken" with SameSite=None; Secure; Partitioned

    8. Refresh the page — user should remain logged in (refresh token works)
    9. Open Chrome Incognito and repeat steps 2-8 (stricter cookie policy)
    10. If step 9 fails (cookie not sent in incognito), this is the known third-party cookie issue — document and plan fallback
