**1. Modify `AuthService.cs`** — add these methods:

`GoogleSignInAsync(string email, string googleId)`: Query user by email. If found, set `GoogleId = googleId` if null, save. If not found, create new User with `Email = email, GoogleId = googleId, EmailConfirmed = true, IsActive = true`. Generate JWT + refresh token, return `AuthResult` (same shape as existing `LoginAsync`).

`RequestMagicLinkAsync(string email)` (per D-10): Find user by email. If not found, return success silently (prevent enumeration). Generate 32-byte random token via `RandomNumberGenerator.GetBytes(32)`. SHA256-hash it. Save `MagicLinkToken { UserId, TokenHash, ExpiresAt = UtcNow.AddMinutes(10) }`. Call `IEmailService.SendMagicLinkAsync(email, $"{frontendUrl}/auth/magic-link?token={Convert.ToBase64String(rawToken)}")`.

`VerifyMagicLinkAsync(string rawTokenBase64)`: Decode base64, SHA256-hash, lookup in `MagicLinkTokens` where `TokenHash` matches, `Used == false`, `ExpiresAt > UtcNow`. If found, set `Used = true`, save. Issue JWT + refresh for `UserId`. Return `AuthResult`.

`SetupTotpAsync(Guid userId)` (per D-02): Generate 20-byte key via `OtpNet.KeyGeneration.GenerateRandomKey(20)`. Encode to base32 via `Base32Encoding.ToString(key)`. Set `user.TotpSecret = base32`. Generate 8 backup codes (each 8-char alphanumeric), SHA256-hash each, store as JSON array in `user.TotpBackupCodes`. Set `user.TotpEnabled = true`. Return `{ secret: base32, backupCodes: plaintext[] }` (plaintext shown once).

`VerifyTotpAsync(string tempToken, string code)` (per D-04): Validate tempToken is a JWT with claim `totp_pending: true` and not expired. Extract userId. Load user. Verify code with `new Totp(Base32Encoding.ToBytes(user.TotpSecret!)).VerifyTotp(code, out _, new VerificationWindow(1, 1))`. If code fails, check backup codes (SHA256 hash input, compare against stored hashes, remove used one). If valid, issue final JWT + refresh. If invalid, return 401.
